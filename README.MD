**HOW TO **


 1. Download files

 ````shell
git clone -b php-fpm8.1 --single-branch https://github.com/vjsdhyygy/caddy-phpfpm-docker caddy-phpfpm-docker 
````

 2. Enter in foder an build

````shell
cd caddy-phpfpm-docker
docker build -t <name> .
````

Or 

````shell
docker pull vjsdhyygy/caddyphp:php81fpm
````
